# Create a title page latex document

Create a title page latex document

## Usage

``` r
create_titlepage_tex(office = "", subdir, species = "")
```

## Arguments

- office:

  primary science center writing the document

- subdir:

  directory where files are going to be held

- species:

  target species for assessment

## Value

Create a \_titlepage.tex document that contains formatting options for a
cover page. The only thing that changes currently is the primary
author's fishery science center.
